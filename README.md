# PsyArXiv Dashboard

An [Observable Framework](https://observablehq.com/framework/) dashboard for [PsyArXiv](https://psyarxiv.com/) preprint metadata and submission information.

## Development

Install requirements:

```
npm install
```

Then, to start the local preview server, run:

```
npm run dev
```

Then visit <http://localhost:3000> to preview your app. For more, see <https://observablehq.com/framework/getting-started>.
